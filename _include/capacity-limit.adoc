Con il modello di licensing basato sulla capacità, ogni sistema Cloud Volumes ONTAP supporta il tiering verso storage a oggetti. La capacità totale suddivisa in livelli può essere scalata fino al limite di bucket del cloud provider. Sebbene la licenza non imponga restrizioni di capacità, dovresti seguire le  https://www.netapp.com/pdf.html?item=/media/17239-tr-4598.pdf["Best practice per FabricPool"^] per garantire prestazioni, affidabilità ed efficienza dei costi ottimali durante la configurazione e la gestione del tiering.
